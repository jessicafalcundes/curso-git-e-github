### criando uma nova branch pelo terminal

**comando necessario para a criacao de uma nova branch pelo terminal:**

**git checkout -b (nome da nova branch)**

## este comando vai criar e já trocar para essa nova branch

## para criar a branch para o github:

**1- acessar repositorio**
**2- ir na seção de listagem das branchs**
**3- inserir nome da branch**
**4- confirmar criacao dessa nova branch**

## caso queira excluir uma branch pelo github

**1- va na seçao de listagem de branchs**
**2- clique na lixeira para excluir**
